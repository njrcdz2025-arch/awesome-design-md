## 使用姿势（给自己看）

1. **接到 UI 任务** → 先在本索引按行业/风格匹配 1-3 个候选品牌。
2. **读完整规范** → `cat resources/awesome-design-md/design-md/<brand>/DESIGN.md`（YAML 或 Markdown 格式，含 color/typography/spacing/component 等完整字段）。
3. **在 prompt 里引用** → "参考 Stripe 的深紫电流+米白画布+Sohne 300 排版" 比 "做个金融风" 有效得多。
4. **组合使用** → 可以取 A 品牌的色板 + B 品牌的排版 + C 品牌的组件语言。
5. **定期更新** → `cd resources/awesome-design-md && git pull`（原仓库仍在活跃维护）。
