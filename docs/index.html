从 DJI 的诗性极简 到 iFlight 的工程荧光—— 26 家品牌划出 5 条清晰的品类语言分界：谁在卖工程、谁在卖诗、谁在卖运动、谁在卖极客、谁在卖全景，一眼可分。

B2B · 工业防务

冷调 + 数据密度 + 工程感

用编辑级排版和实时计数把无人机做成"基础设施年报"。红只做状态灯，不做诱惑。

Skydio Antigravity Autel Robotics
消费单品 · 诗性

大留白 + 中文 letter-spacing + 摄影级

"天地为你所动" / "THE FUTURE IS UP"——用四字诗和苍穹画面把一个 SKU 抬到史诗层。

DJI Global Antigravity Insta360
运动相机 · 标准

黑黄 / 黑金 + 大字 headline + 十宫格购买

高对比 + 按活动分宫格 + 订阅生态——运动相机品类的视觉惯例几乎由 GoPro 一家定义。

GoPro Drift AKASO SJCAM
FPV · 技术极客

白红 / 暗青 / 暖橙 三种极客表达

同一个圈子，三种气质：老练卖场（RadioMaster 红）、工程实验室（iFlight 青）、生活温橙（Flywoo）。

RadioMaster iFlight Flywoo GEPRC BetaFPV
全景相机 · 叙事

PanoX 用国家叙事讲全景（"嫦娥 VR / 火神山"），QooCam 走轻消费，两种截然不同的中国全景表达。

PanoX QooCam Insta360
Off-Category · 混合

安防生态 / D2C 代理 / 农业专线

EZVIZ 走智能家居生态，DJI Ag 用田野绿讲专业农机，D2C 授权店走 Shopify 电商化——非首页型语言。

从 74 家原仓库 fork 而来的本地垂类研究

awesome-design-md 原仓库收录了 74 家全球标杆品牌的设计系统（Stripe / Linear / Notion / DJI 等）。 本项目在 fork 之后，聚焦"无人机 / 相机 / FPV" 一个垂直品类，追加 26 家品牌的首页原始观察，并对其中 9 家最具设计语言差异性的品牌产出完整 DESIGN.md 深度解析，作为本地 vertical study 的补充。

目标：让"给我一份 FPV 品牌官网"这类 UI 生成需求，能在 3 分钟内锁定 1–3 个真实参考。
